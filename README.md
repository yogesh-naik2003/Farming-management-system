# FarmLog AI: Smarter Farming Starts Here 🌾

FarmLog AI is a comprehensive web-based farm management system designed to empower modern farmers with data-driven insights to optimize their operations, boost yields, and increase profitability. This platform provides an intuitive interface to track, manage, and analyze various aspects of farm resources, from natural elements to financial transactions.

## ✨ Features

FarmLog AI offers a wide range of modules to cover all essential aspects of farm management:

### 🌿 Resource Management Categories

-   **Natural Resources**: Manage land plots, monitor soil nutrients, track water sources, log sunlight hours, assess air quality, and record biodiversity observations.
-   **Biological Resources**: Keep inventory of seeds, track crop growth, manage livestock herds, monitor fodder & feed stock, log manure & compost piles, and observe beneficial organisms.
-   **Physical Resources**: Maintain an inventory of machinery, manage irrigation systems, track storage facilities, log transport vehicles, inspect fencing, and oversee farm buildings.
-   **Financial Resources**: Monitor cash flow with income and expense logging, visualize financial data with charts, manage loans & credits, track subsidies & grants, oversee insurance policies, and record capital investments.
-   **Human Resources**: Organize farmer & family roles, manage farm worker rosters, keep track of expert & advisor contacts, and log community group interactions.
-   **Material & Chemical Resources**: Inventory fertilizers, manage pesticides & herbicides, track growth regulators, and monitor packaging materials.
-   **Digital & Knowledge Resources**: Log subscribed software & apps, manage IoT sensors & drones, utilize data services (weather & market info), and track training & knowledge resources.
-   **Energy Resources**: Monitor electricity usage, manage fuel inventory, and track renewable energy assets.

### 🚀 Core Functionality

-   **CRUD Operations**: Easily **C**reate, **R**ead, **U**pdate, and **D**elete data entries across all modules using dynamic modal forms.
-   **Data Persistence**: All your farm data is securely stored in your browser's **Local Storage**, ensuring your information is saved even after closing the browser.
-   **Interactive Dashboard**: A central `homepage.html` provides an overview and navigation to all resource categories.
-   **Responsive Design**: The interface is designed to be accessible and functional across various devices and screen sizes.
-   **Modern UI**: Built with a clean and intuitive user interface using Tailwind CSS and custom styling.
-   **Financial Visualization**: The Financial module includes interactive charts (powered by Chart.js) to help visualize income and expense trends.
-   **CSV Export**: Export your financial transaction log to a CSV file for further analysis or record-keeping.

## 🛠️ Technologies Used

-   **Frontend**:
    -   HTML5
    -   CSS3 (Custom styles, Tailwind CSS)
    -   JavaScript (Vanilla JS for all dynamic functionality)
-   **Libraries/Frameworks**:
    -   Font Awesome (for icons)
    -   Google Fonts (Poppins, Inter)
    -   Chart.js (for data visualization in the Financial module)
    -   Bootstrap 5 (primarily for the landing page `Home.html` navigation and grid system)

## 🚀 Getting Started

Follow these steps to get a local copy of the project up and running.

### Prerequisites

You only need a modern web browser (like Chrome, Firefox, Edge, Safari) to run this project. No server-side setup is required.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone <https://github.com/yogesh-naik-2003/final-hsckshastra.git> # Replace with actual repo URL if different
    cd final-hsckshastra
    ```

2.  **Open in Browser**:
    -   To view the main landing page, open `Home.html` in your preferred web browser.
    -   To directly access the farm management dashboard overview, open `homepage.html`.

## 💡 Usage

1.  **Navigation**: Use the top navigation bar on `homepage.html` to switch between different resource categories (e.g., "Biological", "Natural", "Financial").
2.  **Add New Data**: Within each resource tab, click the "Add New..." button (e.g., "Add New Seed Batch", "Add New Plot") to open a modal form. Fill in the details and click "Save".
3.  **Edit Existing Data**: Locate the entry you wish to modify in the tables or cards, and click the "Edit" button. The modal will pre-populate with existing data for you to update.
4.  **Delete Data**: To remove an entry, click the "Delete" button next to it. A confirmation prompt will appear.
5.  **Financial Overview**: In the "Financial Resources" tab, you can log income and expenses, and view a bar chart summarizing your expenses by category. You can also export your transaction log to a CSV file.
6.  **Data Persistence**: All data you enter is automatically saved in your browser's local storage. This means your data will be available the next time you open the application in the same browser.

# FarmLog AI: Smarter Farming Starts Here 🌾

FarmLog AI is a comprehensive web-based farm management system designed to empower modern farmers with data-driven insights to optimize their operations, boost yields, and increase profitability. This platform provides an intuitive interface to track, manage, and analyze various aspects of farm resources, from natural elements to financial transactions.

## ✨ Features

FarmLog AI offers a wide range of modules to cover all essential aspects of farm management:

### 🌿 Resource Management Categories

-   **Natural Resources**: Manage land plots, monitor soil nutrients, track water sources, log sunlight hours, assess air quality, and record biodiversity observations.
-   **Biological Resources**: Keep inventory of seeds, track crop growth, manage livestock herds, monitor fodder & feed stock, log manure & compost piles, and observe beneficial organisms.
-   **Physical Resources**: Maintain an inventory of machinery, manage irrigation systems, track storage facilities, log transport vehicles, inspect fencing, and oversee farm buildings.
-   **Financial Resources**: Monitor cash flow with income and expense logging, visualize financial data with charts, manage loans & credits, track subsidies & grants, oversee insurance policies, and record capital investments.
-   **Human Resources**: Organize farmer & family roles, manage farm worker rosters, keep track of expert & advisor contacts, and log community group interactions.
-   **Material & Chemical Resources**: Inventory fertilizers, manage pesticides & herbicides, track growth regulators, and monitor packaging materials.
-   **Digital & Knowledge Resources**: Log subscribed software & apps, manage IoT sensors & drones, utilize data services (weather & market info), and track training & knowledge resources.
-   **Energy Resources**: Monitor electricity usage, manage fuel inventory, and track renewable energy assets.

### 🚀 Core Functionality

-   **CRUD Operations**: Easily **C**reate, **R**ead, **U**pdate, and **D**elete data entries across all modules using dynamic modal forms.
-   **Data Persistence**: All your farm data is securely stored in your browser's **Local Storage**, ensuring your information is saved even after closing the browser.
-   **Interactive Dashboard**: A central `homepage.html` provides an overview and navigation to all resource categories.
-   **Responsive Design**: The interface is designed to be accessible and functional across various devices and screen sizes.
-   **Modern UI**: Built with a clean and intuitive user interface using Tailwind CSS and custom styling.
-   **Financial Visualization**: The Financial module includes interactive charts (powered by Chart.js) to help visualize income and expense trends.
-   **CSV Export**: Export your financial transaction log to a CSV file for further analysis or record-keeping.

## 🛠️ Technologies Used

-   **Frontend**:
    -   HTML5
    -   CSS3 (Custom styles, Tailwind CSS)
    -   JavaScript (Vanilla JS for all dynamic functionality)
-   **Libraries/Frameworks**:
    -   Font Awesome (for icons)
    -   Google Fonts (Poppins, Inter)
    -   Chart.js (for data visualization in the Financial module)
    -   Bootstrap 5 (primarily for the landing page `Home.html` navigation and grid system)

## 🚀 Getting Started

Follow these steps to get a local copy of the project up and running.

### Prerequisites

You only need a modern web browser (like Chrome, Firefox, Edge, Safari) to run this project. No server-side setup is required.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone <https://github.com/yogesh-naik-2003/final-hsckshastra.git> # Replace with actual repo URL if different
    cd final-hsckshastra
    ```

2.  **Open in Browser**:
    -   To view the main landing page, open `Home.html` in your preferred web browser.
    -   To directly access the farm management dashboard overview, open `homepage.html`.

## 💡 Usage

1.  **Navigation**: Use the top navigation bar on `homepage.html` to switch between different resource categories (e.g., "Biological", "Natural", "Financial").
2.  **Add New Data**: Within each resource tab, click the "Add New..." button (e.g., "Add New Seed Batch", "Add New Plot") to open a modal form. Fill in the details and click "Save".
3.  **Edit Existing Data**: Locate the entry you wish to modify in the tables or cards, and click the "Edit" button. The modal will pre-populate with existing data for you to update.
4.  **Delete Data**: To remove an entry, click the "Delete" button next to it. A confirmation prompt will appear.
5.  **Financial Overview**: In the "Financial Resources" tab, you can log income and expenses, and view a bar chart summarizing your expenses by category. You can also export your transaction log to a CSV file.
6.  **Data Persistence**: All data you enter is automatically saved in your browser's local storage. This means your data will be available the next time you open the application in the same browser.
